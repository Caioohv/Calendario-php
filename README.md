# Calendario-php


### Sobre o projeto

_CalendAPP_ é uma agenda virtual, onde você pode salvar e visualizar seus eventos, de forma simples, eficiente e segura.

### Tecnologias utilizadas
 - HTML
 - CSS
 - PHP

### Extras
 - Font Awesome

### Utilização Local - Ubuntu

#### Pré requisitos:

 - Lampp (Xampp p/ Linux)
 - MySQL

#### Instruções

Clonar o projeto:

```sh
  $ git clone https://github.com/Caioohv/Calendario-php.git
  ```
Copiar o Projeto para a Pasta _htdocs_

```sh
  $ cp -r Calendario-php/ /opt/lampp/htdocs/
  ```
Iniciar o Apache
```sh
  $ sudo /opt/lampp/xampp start
  ```
Executar o código de criação do banco de dados presente <a href="https://github.com/Caioohv/Calendario-php/blob/main/Database/db.sql">no projeto </a>


Acessar via Browser: localhost/Calendario-php
